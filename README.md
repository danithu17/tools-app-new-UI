# freetools.lk

Initial Next.js app for freetools.lk with a modern dashboard-style homepage focused on PDF tools.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Current scope

- Next.js app router setup with TypeScript
- Dashboard-style landing page with a sidebar, search bar, and optional dark mode
- Base styling ready for the next round of tool pages and flows
